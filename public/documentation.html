MyFlix API

I am building a server side web application for movies. Provides users with access to info about different movies, directors and genres. Users can also sign up and create a whole list of their favorite movies!

Features

Request URL Method Query Parameters Request body data format Response body data format
Return a list of all the movies to the user /movies GET None None A JSON object holding all the data about all the movies
Return data about a movie by title to the user /movies/title GET :movieTitle None A JSON object holding all the data about a single movie. Example:
{ title: 'The Fountain', director: 'Darren Aronofsky', genre: 'Drama' }
Return data about a genre by name movies/genre/genreName GET :genreName None A JSON object format containing movies belonging to a specific genre. Example:
{ title: 'Forrest Gump', director: 'Robert Zemeckis', genre: 'Drama' }
{ title: 'Star Wars: Episode V - The Empire Strikes Back', director: 'Irvin Kershner', genre: 'Science-Fiction' }
Return data about a director by name /movies/director/[director] GET :director None A JSON format data holding all the data about a director by name. Example:
{ name: 'Darren Arnofsky', bio: 'short bio here', birth: '1969' }
Allow users to add a movie to their list of favorites /favourite/add/[id]/[title] POST :add A JSON format data containing all the new data about the movie to be added. Example:
{ title: 'The Fountain', director: 'Darren Aronofsky', genre: 'Drama' } 		A JSON format data containing the newly added movie details.
{ "id": 1, "name": "John Doe", "email": "johndoe@mail.com", "favoriteMovies": [ { title: 'Star Wars: Episode V - The Empire Strikes Back', director: 'Irvin Kershner', genre: 'Science-Fiction' } ] }
Allow new users to register /users/register POST :register Data containing new user details A text response indicating that the registeration was successful
Allow users to update their user info /users/update/[id] PUT :id Data containing changes to old profile A text response indicating that the changes were successfully deployed
Allow users to remove a movie from their list of favorites /favourite/delete/[id]/[title] DELETE :id, :title None An array without the removed movie
Allow existing users to deregister /users/deregister/[id] DELETE :id None A text response indicating that the user profile was successfully deregistered
